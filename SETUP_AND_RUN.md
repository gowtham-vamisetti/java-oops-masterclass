# How to Run the Test Cases Locally\n\n## Prerequisites\n\n1. **Java JDK Installation**\n   - Download from: https://www.oracle.com/java/technologies/downloads/\n   - Verify installation: `java -version` in terminal/command prompt\n\n2. **Text Editor or IDE**\n   - IntelliJ IDEA (Recommended)\n   - Eclipse\n   - VS Code\n   - NetBeans\n   - Or any simple text editor\n\n---\n\n## Method 1: Using Command Line (Windows, Mac, Linux)\n\n### Step 1: Clone the Repository\n```bash\ngit clone https://github.com/gowtham-vamisetti/java-oops-masterclass.git\ncd java-oops-masterclass\n```\n\n### Step 2: Navigate to Level 1 (Classes and Objects)\n```bash\ncd 01-Classes-and-Objects\n```\n\n### Step 3: Compile All Java Files\n```bash\n# Windows\njavac Car.java ClassesAndObjects.java MobilePhone.java MobilePhoneTest.java\n\n# Mac/Linux\njavac *.java\n```\n\n### Step 4: Run the Test File\n```bash\n# Run ClassesAndObjects\njava ClassesAndObjects\n\n# Run MobilePhoneTest\njava MobilePhoneTest\n```\n\n---\n\n## Method 2: Using IntelliJ IDEA (Recommended)\n\n### Step 1: Open Project\n1. Open IntelliJ IDEA\n2. Click **File → Open**\n3. Select the `java-oops-masterclass` folder\n4. Click **Open**\n\n### Step 2: Configure Project SDK\n1. Go to **File → Project Structure**\n2. Under **Project**, set the SDK to your Java JDK\n3. Click **Apply** and **OK**\n\n### Step 3: Run a Test File\n1. Navigate to `01-Classes-and-Objects`\n2. Right-click on `MobilePhoneTest.java`\n3. Click **Run 'MobilePhoneTest.main()'**\n4. View output in the console\n\n### Step 4: Run All Test Files\n```\n01-Classes-and-Objects → MobilePhoneTest.java\n02-Encapsulation → BankAccountTest.java\n03-Inheritance → AnimalSanctuaryTest.java\n```\n\n---\n\n## Method 3: Using VS Code\n\n### Step 1: Install Extension\n1. Open VS Code\n2. Install **Extension Pack for Java** (Microsoft)\n3. Install **Code Runner** (optional)\n\n### Step 2: Open Folder\n1. **File → Open Folder**\n2. Select `java-oops-masterclass`\n\n### Step 3: Run Test File\n1. Open `MobilePhoneTest.java`\n2. Click **Run** button (top right) or press `Ctrl+Alt+N`\n3. View output in terminal\n\n---\n\n## Quick Test: Level-by-Level\n\n### Level 1: Classes and Objects\n```bash\ncd 01-Classes-and-Objects\njavac MobilePhone.java MobilePhoneTest.java\njava MobilePhoneTest\n```\n\n### Level 2: Encapsulation\n```bash\ncd 02-Encapsulation\njavac BankAccount.java BankAccountTest.java\njava BankAccountTest\n```\n\n### Level 3: Inheritance\n```bash\ncd 03-Inheritance\njavac AnimalClass.java Lion.java Elephant.java Penguin.java AnimalSanctuaryTest.java\njava AnimalSanctuaryTest\n```\n\n---\n\n## Troubleshooting\n\n### Error: \"command not found: javac\"\n**Solution:**\n- Java is not installed or not in PATH\n- Download and install JDK from Oracle\n- Add Java bin folder to system PATH\n\n### Error: \"cannot find symbol\"\n**Solution:**\n- Make sure all files are in the same directory\n- Check for typos in class names\n- Recompile all files: `javac *.java`\n\n### Error: \"Exception in thread 'main'\"\n**Solution:**\n- Check file paths\n- Ensure main method exists in test file\n- Verify class names match filenames\n\n---\n\n## All Test Files Summary\n\n| Level | Test File | Command |\n|-------|-----------|----------|\n| 1 | MobilePhoneTest | `java MobilePhoneTest` |\n| 2 | BankAccountTest | `java BankAccountTest` |\n| 3 | AnimalSanctuaryTest | `java AnimalSanctuaryTest` |\n| 4 | (To be created) | - |\n| 5 | (To be created) | - |\n\n---\n\n## Expected Output Examples\n\n### Level 1 Output:\n```\n=== Apple iPhone 13 Specifications ===\nBrand: Apple\nModel: iPhone 13\nPrice: $999.99\nRAM: 4 GB\nStorage: 128 GB\n\nAfter 10% discount: $899.99\nIs affordable within budget of $900? true\n```\n\n### Level 2 Output:\n```\n=== Account Statement ===\nAccount Number: ACC001\nAccount Holder: John Doe\nAccount Type: Savings\nBalance: $1000.0\n\n✓ Deposit successful! Check Java installation: `java -version`\n2. Verify files are in correct directory: `ls` or `dir`\n3. Check for compilation errors: `javac -verbose *.java`\n4.
